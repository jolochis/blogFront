# Descripción

Aplicacion creada con vite, typescript y tailwind

## Instrucciónes

instalar modulos

```
$ docker build -t blogfront .  
```

Ejecutar aplicación

```
$ docker run -p 5173:5173 --name blogfront blogfront
```
